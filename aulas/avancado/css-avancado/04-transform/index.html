Transform

Aplicando transformações com transform

1- Transform Scale

Utilizando o Scale, podemos determinar que a imagem seja multiplicada pelos valores passados, sendo para os eixos X e Y. o primeiro exemplo, informamos em único código, nos exemplos 2 e 3, passamos um por vez.

  • img.scale{
    transform: scale(2, 3);
    }
  • img.scale{
    transform: scaleX(2);
    }
  • img.scale{
    transform: scaleY(3);
    }

2- Transform Rotate

Utilizando o Rotate, vamos informar o quanto queremos rotacionar a imagem, sempre sendo em degraus (deg).
Podemos rotacionar em um dos eixos, porém não será tão exato, pois para rotacionar o correto é sobre os dois eixos

  • img.scale{
    transform: rotate(90deg);
    }
  • img.scale{
    transform: rotateX(20deg);
    }
  • img.scale{
    transform: rotateY(15deg);
    }

3- Transform Translate

Utilizando Translate faz uma translação da imagem, em 2D.
A imagem será deslocada para o ponto informado entre parênteses, podendo ser (X, Y) ou um por vez, ou somente 1.
A unidade utilziada é "px"

  • img.scale{
    transform: translate(100px, 15px);
    }
  • img.scale{
    transform: translateX(100px);
    }
  • img.scale{
    transform: translateY(15px);
    }
