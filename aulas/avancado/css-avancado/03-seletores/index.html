Estilização com duas Classes

Título 1

Incluindo duas classes no mesmo elemento, ou mais, nos permite criar estilizações específicas ao adicionar, no css, uma classe após a outra sem espaçamento.

  • .primeiraClasse.segundaClasse {
    estilo: valor;
    }
  • .primeiraClasse.outraClasse {
    estilo: outroValor;
    }

Título 2

Dessa forma, ao colocar alguma estilização para a primeiraClasse, será aplicada para todos.

Estilização com seletor >

Esta estilização, permite estilizar seletores específicos dentro de outros seletores.

Neste exemplo, estamos estilizando as tags "p" dentro da "div", já as outras tags, "ul"/"li"/"span", não irão receber os parâmentros setados.
São consideradas filhas dentro da "div", porém, não foram selecionadas.

  • seletor > outroSeletor {
    estilo: valor;
    }
  • Os outro seletores dentro do seletor pai, não irão receber o estilo, pois não foram selecionados.

Estilização com seletor +

Agora vamos estilizar uma tag que vem logo após outra específica.

Vamos estilizar da seguinte forma

div + p {
color: yellow;
background-color: black;
}

Como estamos vendo, só o primeiro "p" foi estilizado com a cor amarela, estes dois próximos, não receberam esta estilização.
